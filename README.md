# Olist Delivery Time Forecasting

This project uses the Olist e-commerce dataset to predict average delivery time for new customers using Python and machine learning.

## Project Objective
The goal of this project is to forecast delivery time and generate business insights that can support logistics planning and improve customer experience.

## What I Did
- Cleaned and merged multiple e-commerce datasets
- Conducted exploratory data analysis (EDA)
- Performed feature engineering and multicollinearity checks
- Built and compared multiple regression and machine learning models
- Tuned models using Optuna
- Evaluated performance and conducted error analysis
- Generated business recommendations based on model results

## Models Used
- Linear Regression
- Lasso Regression
- Random Forest
- XGBoost
- LightGBM
- Stacking Ensemble
